### Bilingual Content Support

Every story in Casa de Historias is presented in both Spanish and Nahuatl:

- **Spanish (Español)** - Accessible to wider Spanish-speaking audiences
- **Nahuatl** - Preserves the indigenous language and authentic cultural expression

Users can seamlessly switch between languages to compare translations or learn vocabulary.

### Story Creation & Editing

Authenticated users can contribute their own stories:

<Steps>
  <Step title="Add Story Screen">

  </Step>
  
  <Step title="Enter Bilingual Content">

### Favorites System

Users can save stories to their personal favorites collection:

```kotlin FirestoreRepository.kt:179
suspend fun addToFavorites(userId: String, storyId: String): Boolean {
    return try {
        firestore.collection("users").document(userId)
            .update("favoriteStories", 
                com.google.firebase.firestore.FieldValue.arrayUnion(storyId))
            .await()
        true
    } catch (e: Exception) {
        false
    }
}
```

Favorites are:
- Synced to Firebase for access across devices
- Available offline via Room database
- Managed through the `FavoritesViewModel` and `FavoritesScreen`

- **Coil** for efficient image loading with caching

- **ProGuard** enabled for release builds to reduce APK size
