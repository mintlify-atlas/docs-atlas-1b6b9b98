## Welcome to Casa de Historias

Casa de Historias is a mobile application designed to preserve and share the rich cultural heritage of indigenous legends from the Sierra de Zongolica region. Built with Kotlin and Jetpack Compose, it provides a bilingual platform for storytelling in both Spanish and Nahuatl.

## Built With Modern Technology

Casa de Historias leverages cutting-edge Android development tools and frameworks:

- **Jetpack Compose** for modern, declarative UI
- **Firebase** for authentication, data storage, and real-time sync
- **Room Database** for offline-first data persistence
- **Hilt** for dependency injection
- **MapLibre** for interactive mapping
- **Media3 ExoPlayer** for audio playback
