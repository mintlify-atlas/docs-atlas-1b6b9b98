## Overview

The `FavoritesViewModel` class manages the user's favorite stories list and provides functionality to toggle favorites. It maintains both the full story objects and a set of favorite IDs for quick lookups.

## Methods

### loadFavorites

```kotlin
fun loadFavorites()
```

Loads the current user's favorite stories from Firestore.

**Side effects:**
- Sets `isLoading` to true during loading
- Updates `favoriteIds` with the user's favorite story IDs
- Updates `favorites` with full story objects
- Sets `isLoading` to false when complete
- If user is not authenticated, sets empty lists

**Source:** `~/workspace/source/app/src/main/java/com/example/casadehistorias/ui/favorites/FavoritesViewModel.kt:34`

### toggleFavorite

```kotlin
fun toggleFavorite(storyId: String)
```

Toggles a story's favorite status. If the story is currently favorited, it removes it; otherwise, it adds it.

<ParamField path="storyId" type="String" required>
  The unique identifier of the story to toggle
</ParamField>

**Side effects:**
- Updates the user's favorites in Firestore
- Immediately updates local state (`favoriteIds` and `favorites`) for responsive UI
- If adding: fetches story data and appends to favorites list
- If removing: filters story from favorites list

## Usage Example

```kotlin
@Composable
fun FavoritesScreen(
    viewModel: FavoritesViewModel = hiltViewModel()
) {
    val favorites by viewModel.favorites.collectAsState()
    val isLoading by viewModel.isLoading.collectAsState()

    if (isLoading) {
        LoadingIndicator()
    } else if (favorites.isEmpty()) {
        EmptyFavoritesView()
    } else {
        LazyColumn {
            items(favorites) { story ->
                StoryCard(
                    story = story,
                    isFavorite = true,
                    onFavoriteClick = {
                        viewModel.toggleFavorite(story.id)
                    }
                )
            }
        }
    }
}

@Composable
fun StoryCard(
    story: Story,
    viewModel: FavoritesViewModel = hiltViewModel()
) {
    val isFavorite = viewModel.isFavorite(story.id)
    
    Card {
        // Story content...
        IconButton(
            onClick = { viewModel.toggleFavorite(story.id) }
        ) {
            Icon(
                imageVector = if (isFavorite) {
                    Icons.Filled.Favorite
                } else {
                    Icons.Outlined.FavoriteBorder
                }
            )
        }
    }
}
```

## Optimistic Updates

The `toggleFavorite` method implements optimistic updates - the local state is updated immediately before the Firestore operation completes, providing a responsive user experience:

```kotlin
if (isFav) {
    storyRepository.removeFromFavorites(storyId)
    _favoriteIds.value = _favoriteIds.value - storyId
    _favorites.value = _favorites.value.filter { it.id != storyId }
} else {
    storyRepository.addToFavorites(storyId)
    _favoriteIds.value = _favoriteIds.value + storyId
    // Fetch and add story to list...
}
```
