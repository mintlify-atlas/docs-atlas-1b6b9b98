## Overview

The `AddStoryViewModel` class manages the creation and updating of stories, including image and audio file uploads to Firebase Storage. It provides UI state management for the story submission process.

## Usage Example

```kotlin
@Composable
fun AddStoryScreen(
    viewModel: AddStoryViewModel = hiltViewModel(),
    onNavigateBack: () -> Unit
) {
    val uiState by viewModel.uiState.collectAsState()
    
    LaunchedEffect(uiState) {
        when (uiState) {
            is AddStoryUiState.Success -> {
                // Show success message
                onNavigateBack()
            }
            is AddStoryUiState.Error -> {
                // Show error message
            }
            else -> {}
        }
    }

    AddStoryForm(
        onSubmit = { formData ->
            viewModel.addStory(
                titleEs = formData.titleEs,
                titleNahuatl = formData.titleNahuatl,
                contentEs = formData.contentEs,
                contentNahuatl = formData.contentNahuatl,
                narratorName = formData.narratorName,
                community = formData.community,
                imageUri = formData.imageUri,
                tags = formData.tags
            )
        },
        isLoading = uiState is AddStoryUiState.Loading
    )
}
```

## File Upload Process

The ViewModel handles file uploads in the following order:

1. Generate temporary story ID using `UUID.randomUUID()`
2. Upload image to Firebase Storage (if provided)
3. Get download URL from uploaded image
4. Create Story object with image URL
5. Save story to Firestore
6. Return final story ID

```kotlin
val tempId = UUID.randomUUID().toString()
var uploadedImgUrl: String? = null
if (imageUri != null) {
    uploadedImgUrl = storageRepository.uploadImage(imageUri, tempId)
}
```

## Error Handling

Errors are captured and exposed through the `uiState`:

```kotlin
val resultId = repository.addStory(newStory)
if (resultId != null) {
    _uiState.value = AddStoryUiState.Success
} else {
    _uiState.value = AddStoryUiState.Error(
        "No se pudo guardar la historia. Verifica tu conexión."
    )
}
```
