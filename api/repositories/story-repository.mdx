## Overview

The `StoryRepository` class provides a clean interface for accessing and managing story data. It coordinates between `FirestoreRepository` and `AuthRepository`, handling data transformation between Firebase models and domain models.

## Declaration

```kotlin
@Singleton
class StoryRepository @Inject constructor(
    private val firestoreRepository: FirestoreRepository,
    private val authRepository: AuthRepository
)
```
