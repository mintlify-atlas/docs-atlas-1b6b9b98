## Overview

The `Story` data class is the primary domain model representing a story in the application. It contains bilingual content (Spanish and Nahuatl), multimedia URLs, location data, and metadata.

## Declaration

```kotlin
data class Story(
    val id: String,
    val titleEs: String,
    val titleNahuatl: String,
    val contentEs: String,
    val contentNahuatl: String,
    val audioUrl: String?,
    val imageUrl: String?,
    val narratorName: String,
    val community: String,
    val latitude: Double? = null,
    val longitude: Double? = null,
    val authorId: String = "",
    val tags: List<String> = emptyList(),
    val createdAt: Long = 0L
)
```

## Usage Example

```kotlin
val story = Story(
    id = "story123",
    titleEs = "La leyenda del volcán",
    titleNahuatl = "Popocatepetl itlahtol",
    contentEs = "Había una vez...",
    contentNahuatl = "Ye huehcauh...",
    audioUrl = "https://storage.googleapis.com/audio/story123.mp3",
    imageUrl = "https://storage.googleapis.com/images/story123.jpg",
    narratorName = "María González",
    community = "Tepoztlán",
    latitude = 18.9847,
    longitude = -99.0930,
    authorId = "user456",
    tags = listOf("leyenda", "volcán", "tradición"),
    createdAt = 1709510400000L
)
```
